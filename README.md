The Art of C++ / Tuple is a C++11 header-only reference implementation of [`std::tuple`](http://en.cppreference.com/w/cpp/utility/tuple).

## Compatibility

* Requires C++11 or newer.
* Tested with GCC 4.8+ and Clang 3.4+.
